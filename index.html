Configuration Google Calendar

Étape 1 — Créer un projet Google Cloud
1. Va sur console.cloud.google.com
2. Crée un projet (ou utilise un existant)
3. Active l'API Google Calendar API (menu API → Bibliothèque)
4. Va dans Identifiants → Créer → ID client OAuth 2.0
5. Type : Application Web
6. Ajoute ton URL GitHub Pages dans "Origines JavaScript autorisées"
7. Copie le Client ID ci-dessous
Étape 2 — Coller ton Client ID
Laisse vide pour utiliser ton calendrier principal
